WooCommerce Composite Prodct Plugin

Enable your to add multiple products as a single product to cart at a time

Thank you for purchasing my plugin. If you have any questions that are beyond the scope of this help file.

Overview

Allow customers to create the product bundle in the step by step selection of items from specific products, or categories, the admin can force the customers to specific limit of selection of items from each step. The customers can go for each step & create the bundle of their own choice.

Minimum requirements

  • PHP 5.4
  • WordPress 4
  • WooCommerce 2.4
  • JavaScript / jQuery

Sources

  • WordPress
  • WooCommerce

Installation

There are two primary methods for installing any WordPress plugin.

Install with a zip file

  1. From your WordPress dashboard, choose Plugins > Add New
  2. Select Upload from the set of links at the top of the page (the second link)
  3. From here, browse for the zip file included in your plugin purchase titled composite-product.zip and click the Install Now button
  4. Once installation is complete, click Activate Plugin to activate it and enable its features.

Install by FTP Manager

If you are have no previous web experience and have no idea what an FTP Manager is, then this option is not for you.

  1. Access your host through the FTP manager
  2. Access the plugins directory of your WordPress install /(WordPress directory)/wp-content/plugins/

    If WordPress is installed on the root folder the directory is /wp-content/plugins/

  3. Find the directory titled show product variations and upload it and all files within to the directory from the previous step.
  4. From your WordPress dashboard, choose Plugins > Installed Plugins
  5. Locate the newly added plugin and click on the Activate link to enable its features.

Quick Start Guide

WooCommerce Composite Prodct Plugin is really easy to use. Below we described it step by step

Basic plugin usage

Step 1

Enable module from general options.

Step 2

Enable/disable more option according to your product. Formate the boxes.

Step 3

Create a composite product with maximum no of boxes and steps.

Choose products from every step and add to box.

When added all products, review your added products. So then you have to edit the products back if required or add them to cart.

Backend Options

There are several options on admin panel of WooCommerce Composite Prodct Plugin. You can set them one by one.

General Settings

General settings tab is the main options area of plugin.

Enable Module

You can enable / disable composite product functionality from here

Scroll up

When you add product to box it will scrollup. You can enable/disable it by checking this option.

Show Variable Products

You can allow variable product variations by checking this option

Cart Button Label

You can set cart button text from here.

Add to Box Button Label

You can set add to box button text from here.

Product Boxes Background Color

You can set composite boxes background color from here

Product Boxes Border Color

You can set composite boxes border color from here

Product Options

You can set composite product here

First you have to select your product type 'Composite Product'

Then you will be able to see the product boxes tab in side menu. By clicking product boxes tab you will get more information about it.

Pricing Type

You can set composite product pricing type from here. There are three types of pricing types.

Product Addons Price Only: Your product initial price will be zero and it will count only added products prices.

Product Addons Price + Regular price: Your product initial price will be equal to its regular price and it will add price of its added products also.

Fixed price: Your product price will be eual to regular price.

Box Layout Columns

You can set columns of your boxes it may 3, 4, 5, 6 etc.

Products Layout Columns

You can set columns of your products of the composite steps it may 3, 4, 5, 6 etc.

Show Products Per Page

You can set to show how many products initially so that you can load more products by loader.

Enable Product Price

You can enable/disable product prices by checking this option.

Add New Box

You can Add Boxes by clicking this button.

Boxes Options

You can also set your box individually.

You can set box name, Box title, Box short description, .

you can set No of Composite Product Boxes.

Add minimum Products to Boxes: you can validate your box by his option. So you will be restrict of adding products to box with limit.

You can choose as many products as you want by select 2 dropdown and categories also.

Frontend View

Composite Product

You can see the boxes with label where products will be added. its footer section contain price and navigation system.

You can see the products which you have to add to box with custom button of add to box. You cal also see load more button to load more products of the same step.

When product added to box. So you can remove the added product by clicking on box. You can see the navigation button which will be enabled/disabled according to limit of box.

When all products added to all steps so you will be redirected to review your added products. So from here you can edit them also or add them to cart.
